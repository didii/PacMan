Detailed Description

Queue a single movement with some extra functionality.

Needs a Rigidbody2D attached to the GameObject. It updates the velocity of it automatically.

Utility.EDirection4 MoveQueue.NextDirection
getset

Gets or sets the next direction of movement. CurrentDirection will be updated if allowed.

CurrentDirection is immediately updated if Wait was not recently used and

  • CurrentDirection is set to Utility.EDirection4.None
  • value is the opposite of CurrentDirection when AllowReverse was specified

In all other cases when Wait was not recently used, value is discarded if it is the same as CurrentDirection and stored if not.

If Wait was recently used, CurrentDirection will always immeidately update if the direction is allowed.
