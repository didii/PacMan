void SyncTimer.Start ( )

Resets and starts the timer by raising the Elapsed event every IntervalTime seconds for RepeatCount times.

Set RepeatCount = 0 to repeat the event indefinately.

See also
IgnoreMultipleStart, RepeatCount, InitialRaise

bool SyncTimer.Enabled
getset

Enable or disable the raising of Elapsed.

bool SyncTimer.IgnoreMultipleStart
getset

Ignore a call to Start when the timer was already started. If not, the timer is reset every time when called.

bool SyncTimer.InitialRaise
getset

Set to true to immediately raise Elapsed on Start.

float SyncTimer.IntervalTime
getset

How much time should be between the event calls.

bool SyncTimer.Repeat
getset

Set true to restart the timer after Elapsed has been triggered.

When set to false and InitialRaise set to true, the event will fire twice. Once when Start was called and once after IntervalTime seconds.

int SyncTimer.RepeatCountMax
getset

How many times should Elapsed be raised.
